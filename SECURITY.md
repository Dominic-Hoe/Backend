# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

If you do find a vulnerability in this repository (whether it be an authentication vulnerability, dependency vulnerability et cetera), feel free to report it as an issue.
